# Sistema de Chamados

Projeto desenvolvido para a disciplina de Linguagem de Programação IV.

## Tecnologias

- PHP
- MySQL
- Bootstrap 5
- JavaScript
- Chart.js
- Arquitetura MVC

## Funcionalidades

- Cadastro de chamados
- Listagem de chamados
- Pesquisa por cliente
- Edição de chamados
- Exclusão de chamados
- Dashboard com gráfico
- Estatísticas por status
